# Bank Loan Analysis Dashboard (Excel Project)

## Project Overview

This project analyzes bank loan data using Microsoft Excel to understand loan distribution, customer risk levels, and loan approval trends.
The dashboard helps visualize key metrics related to loan performance and customer profiles.

## Dataset

The dataset contains **500 loan records** with the following key fields:

* Loan ID
* Customer ID
* Age
* Credit Score
* Employment Type
* Salary Band
* Loan Amount
* Loan Status
* Risk Level
* Region

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design

## Key Dashboard Metrics

The dashboard provides insights into:

* Total Loans Issued
* Total Loan Amount
* Active Loans
* Default Rate
* Average Credit Score

## Dashboard Visualizations

The Excel dashboard includes:

* Loan Amount by Region
* Loan Status Distribution
* Loan Purpose Analysis
* Monthly Loan Trend
* Risk Level Distribution
